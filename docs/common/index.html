Components Used in Both Survey and Design

Design and Survey Share a few components:

CommonMenu

js/src/commoncomps/CommonMenu.js

Present on all pages, contains the header, logo, link to homepage and sign out button

CommonAuthGate

js/src/commoncomps/CommonAuthGate.js

If the user is logged in, renders

  • DesignAuthGate if the url starts with “/design”

  • DesignAuthGate if the url starts with “/survey”

  • CommonChooseView if the url does not start with design or survey

If the user is logged out, renders

  • CommonLoggedOut

CommonChooseView

js/src/commoncomps/CommonChooseView.js

[To be implemented] queries the permissions collecction for the current users email.

If the user is marked ‘admin’

  • Presents links to /design and /survey

Otherwise, redirects to

  • /survey

CommonLoggedOut

js/src/commoncomps/CommonLoggedOut.js

Splash page with link to Google Sign In, post sign in, redirects to /
